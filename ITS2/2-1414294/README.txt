Accessing your data
-------------------

Access to your data is available via the JGI portal using your JGI login,
if you have not yet registered, please do so at:
https://signon.jgi-psf.org/register. Once you have access the portal
website, you will be able to download the itags data and associated
analysis files. PIs automatically have access to the portal data. You may
grant access to collaborators by going to the admin page (next to the
Download button), clicking access control, enter the collaborators email
address into the first green box then go to the 'actions' section and click
'grant.' The system will send an automated email to the person. If this
person cannot be found in the system, they will also need to create a JGI
account using the above link.

Output directories are compressed as .tar.bz2 files, these can be opened
normally on Mac OS X. On a windows computer you may need additional software,
e.g. WinZip, to open these.
